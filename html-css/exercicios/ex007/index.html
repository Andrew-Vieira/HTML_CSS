Principais formatações

Negrito/ Destaque

Nesta frase temos um termo em negrito usando a tag STRONG

Itálico / Ênfase

Nesta frase temos um termo em ênfase (Itálico) usando a tag EM

Texto marcado

Podemos criar também um texto marcado usando a tag MARK, (para alterar a cor da marcação utilizamos o CSS)

Texto deletado

Podemos também marcar um texto como excluido dessa forma aqui, utilizando DEL

Texto inserido

Podemos também marcar um texto como inserido para dar uma enfase (sublinhar basicamente), utilizando INS

Texto sobrescrito

Para inserir coisas do tipo x20+3, basta utilizar a tag SUP

Texto subscrito

Para inserir coisas do tipo h2o, basta utilizar a tag SUB
